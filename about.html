努力过自己想过的生活。

我是梁雨，一个喜欢折腾的工科生，热爱互联网技术和数据科学，喜欢尝试，喜欢分享，喜欢有创造力的工作，目前的研究方向是工业数据挖掘和机器学习，这个博客主要是记录自己的学习和生活。

我不是计算机科班出身，刚开始接触机器学习和数据挖掘的时候感觉无从下手，看了很多技术大牛的博客，从他们那学到了很多，所以也想把自己的知识和经验写出来，希望有人能看到，希望能对你有帮助。

曾经尝试利用 Python+Django 搭建个人博客，由于一些原因搁置了，后来看到有人用 GitHub Pages + Jekyll 搭建个人博客，在Github上Fork了@Huxpro 的博客项目，在他的基础上仅仅做了很小的修改，真的很惭愧，希望以后能有比较大的改版，也很感谢 GitHub 给了我一个免费搭建个人博客的机会，开源改变世界！
